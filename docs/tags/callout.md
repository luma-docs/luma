# Callout

Use callouts to draw attention to important information, such as warnings, tips, or 
additional context, by presenting them in highlighted rectangular boxes that stand out 
from regular text.

## Callouts without titles

``` {% process=false %}
{% callout %} 
This is a callout.
{% /callout %}
```

{% callout %} 
This is a callout.
{% /callout %}

## Callouts with titles

``` {% process=false %}
{% callout %} 
This is a callout.
{% /callout %}
```

{% callout title="Title" %} 
This is a callout with a title.
{% /callout %}
